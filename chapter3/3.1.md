# 配置文件
每个文档都需要在文档目录创建一个```config.json```文件，目的是为了与在线的文档相关联。 示例：
```
{  
    "name": "gitbook使用教程",  
    "description": "该文档介绍的了怎样通过gitbook来编写文档。",  
    "group": "该文档所在的项目",  
    "sorted": 5  
}  
```
**name**：文档的名称  
**description**：文档描述  
**group**：文档所在的项目,可以用项目名称，比如：shopify，erp。  
**sorted**：文档排序，序号小的在前。
