Gitbook 使用教程

安装gitbook

gitbook 的安装非常简单，详细指南可以参考 gitbook 文档。

安装要求

NodeJS (v4.0.0版本以上)

NodeJS下载地址：

https://nodejs.org/en/download/

支持平台

Windows, Linux, Unix, or Mac OS X

安装nodeJs后，通过npm安装

$ npm install gitbook-cli -g
